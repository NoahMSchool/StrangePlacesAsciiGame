# A Strange Place It Is ASCII
**ASCII**

This is a web based game submitted for the 2026.1 brackeys game jam which had a theme called strange places.
There were two contributers to this project.
We tried to do something different for this jam. It is a text based adventure that uses ASCII art.

The languages used are HTML, CSS and Javascript.
